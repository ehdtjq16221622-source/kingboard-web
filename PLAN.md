# Kingboard Web - 계획서

## 프로젝트 개요
- **서비스명:** Kingboard 말투 변환기
- **목적:** 데스크탑에서 말투 변환 서비스 제공 + 앱 유입 채널 + 광고 수익
- **타겟:** 직장인, 소개팅 앱 사용자, 학부모 등 말투에 신경 쓰는 사람

---

## 기술 스택
| 항목 | 선택 | 이유 |
|------|------|------|
| 프론트엔드 | HTML/CSS/JS (바닐라) | 빠른 로딩, SEO 유리, 배포 간단 |
| 백엔드 | Supabase Edge Functions | API 키 보호, 무료 플랜 충분 |
| 호스팅 | Vercel | 무료, GitHub 연동, CDN |
| 광고 | Google AdSense | 설치 간단, CPC 방식 |
| 도메인 | 별도 구매 권장 (kingboard.kr 등) | 신뢰도 + SEO |

---

## 파일 구조
```
kingboard-web/
├── PLAN.md              ← 이 파일
├── index.html           ← 메인 페이지
├── style.css            ← 스타일
├── script.js            ← 변환 로직 + API 호출
└── supabase/
    └── functions/
        └── convert/
            └── index.ts ← Edge Function (GPT API 호출)
```

## 말투 옵션
1. **적당한 존댓말** - 기본 정중체
2. **엄격 격식체** - 공식 문서/이메일
3. **사내 메시지** - 직장 내 메신저
4. **고객 응대** - 친절한 고객 서비스
5. **학부모 안내** - 학교/학원 공지
6. **소개팅** - 자연스럽고 호감 가는 말투

## Supabase Edge Function 구조

### 엔드포인트
`POST /functions/v1/convert`

### 요청
```json
{
  "text": "원문 텍스트",
  "style": "고객응대",
  "mode": "tone"
}
```

### 응답
```json
{
  "result": "변환된 텍스트",
  "tokensUsed": 150
}
```

### Rate Limiting
- IP당 하루 10회 무료
- 초과 시 앱 다운로드 유도 메시지 표시

## 광고 전략

### AdSense 배치 (3곳)
| 위치 | 사이즈 | 예상 노출 |
|------|--------|----------|
| 상단 배너 | 728x90 Leaderboard | 모든 방문자 |
| 우측 사이드바 | 300x250 Rectangle | 변환 사용자 |
| 하단 배너 | 728x90 Leaderboard | 결과 확인 후 |

### AdSense 신청 조건
- 도메인 필요 (GitHub Pages도 가능하나 커스텀 도메인 권장)
- 콘텐츠 충분해야 함 (서비스 설명, 사용법 등 추가)
- 승인까지 1~2주 소요

---

## 배포 순서

1. **지금:** 웹사이트 파일 완성
2. **Supabase 설정:**
   - 프로젝트 생성
   - `OPENAI_API_KEY` 환경변수 등록
   - Edge Function 배포: `supabase functions deploy convert`
   - `config.js`에 Supabase URL + anon key 입력
3. **GitHub 업로드 + Vercel 연동**
4. **AdSense 신청**
5. **도메인 연결** (선택)
